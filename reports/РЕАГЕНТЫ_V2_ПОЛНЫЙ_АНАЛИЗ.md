# Реагенты V2: Расчет по фактическим данным ЛИС, Applogs, errors.xml и складу

## 1. КОНЦЕПЦИЯ: ДВА ПОТОКА УЧЕТА

### Поток 1 — Легитимные тесты (из ЛИС с направлением)
Источник: сводные данные Damumed. Расчет: `тесты × норма расхода = ожидаемый расход`

### Поток 2 — "Левые" движения (без направления)
Источник: Applogs (`THERE IS NO SR FOR Barcode`) + errors.xml (пробы без соответствия в ЛИС)

### Особый случай: легитимный rerun
Лаборант перепроверяет аномальный результат с рандомным штрих-кодом.
**Детекция**: если в ±30 мин после легитимного теста на том же анализаторе есть "левый" прогон с похожим Test Mode → вероятный RERUN. Норма reruns: 2-5%. Если "левых" >10% — аномалия.

### Итоговая формула
```
СКЛАД (закуплено) - ПОТОК 1 (легитимные) - СЕРВИСНЫЕ (QC/калибр/startup) = ОЖИДАЕМЫЙ ОСТАТОК
ОЖИДАЕМЫЙ ОСТАТОК - ФАКТИЧЕСКИЙ ОСТАТОК = РАСХОЖДЕНИЕ
Из логов: ПОТОК 2 × норма расхода ≈ должно совпасть с РАСХОЖДЕНИЕМ
```

## 2. ВСЕ 15 ЕДИНИЦ ОБОРУДОВАНИЯ

### 2.1 BC-5000 — Гематологический анализатор
**Реагенты M-52**: Diluent (20л), DIFF Lyse (500мл×4), LH Lyse (100мл×4), Probe Cleanser (50мл)

| Операция | Diluent мл | DIFF мл | LH мл | Cleanser мл |
|----------|-----------|---------|-------|-------------|
| **1 тест CBC+DIFF** | **27.5** | **1.0** | **0.2** | 0 |
| 1 тест CBC only | 23.2 | 0 | 0.2 | 0 |
| Startup | 65 | 1 | 0.2 | 0 |
| Shutdown | 60.9 | 1 | 0.2 | 2 |
| QC (1 уровень) | 27.5 | 1 | 0.2 | 0 |
| Exit Standby <1ч | 3.7 | 0 | 0 | 0 |
| Exit Standby 1-5ч | 16.9 | 0 | 0 | 0 |
| Exit Standby >5ч | 65 | 3 | 0.2 | 0 |
| Калибровка (12 прогонов) | 330 | 12 | 2.4 | 0 |

Из 20л Diluent ≈ **727 тестов**. Из 2000мл DIFF ≈ **2000 тестов**. Из 400мл LH ≈ **2000 тестов**.

**Пробирка**: ЭДТА (сиреневая крышка), 2-3 мл крови.
**USB-выгрузка**: Service → Advanced Toolbox → One-key Export (FAT32 флешка). Содержит counter info + все результаты + error log.

### 2.2 BS-240 — Биохимический анализатор
Минимальный реакционный объем: **100 мкл**. Каждый тест = свой реагент (R1 + R2 + проба).

| Тест | R1 мкл | R2 мкл | Проба мкл |
|------|--------|--------|-----------|
| АЛТ | ~180 | ~45 | ~5 |
| АСТ | ~180 | ~45 | ~5 |
| Глюкоза | ~180 | — | ~3 |
| Общий белок | ~250 | — | ~4 |
| Мочевина | ~180 | ~45 | ~3 |
| Креатинин | ~180 | ~45 | ~10 |
| Билирубин | ~180 | ~45 | ~5 |
| Амилаза | ~180 | ~45 | ~4 |
| СРБ кол-во | ~150 | ~50 | ~3 |
| Холестерин | ~250 | — | ~3 |

> Точные объемы ОБЯЗАТЕЛЬНО взять из вкладышей ваших лотов реагентов — различия ±20%.

**Пробирка**: сыворотка (красная/желтая крышка) или литий-гепарин (зеленая).
80 реагентных позиций, до 200 тестов/час.

### 2.3 BS-240Pro — Биохимический анализатор
Аналогичен BS-240. Те же реагенты, те же объемы. Совместимая система.

### 2.4 BS-430 — Биохимический анализатор
Те же реагенты что BS-240. До 400 тестов/час. Объемы на тест идентичны.

### 2.5 BS-600M — Биохимический анализатор
Те же реагенты. До 600 тестов/час. Может иметь ISE-модуль (Na/K/Cl — отдельные реагенты).

### 2.6 C3100 Mindray — Коагулометр
**Детекция**: магнитная (вискозиметрическая). **Тесты**: PT, APTT, FIB, TT, D-Dimer, факторы.

| Тест | Реагент | ~Объем реагента | ~Объем пробы | Примерно тестов из набора |
|------|---------|----------------|-------------|--------------------------|
| PT | Тромбопластин | ~100 мкл | ~50 мкл | ~500 из 50мл |
| APTT | APTT-реагент + CaCl₂ | ~100+100 мкл | ~50 мкл | ~500 |
| FIB | Тромбин | ~100 мкл | ~100 мкл | ~250 |
| TT | Тромбин | ~100 мкл | ~100 мкл | ~250 |

**КРИТИЧНО**: Лиофилизированные реагенты после растворения живут **4-8 часов**. Остатки — потеря.
**Пробирка**: цитратная (голубая крышка, 3.2% натрия цитрат).

### 2.7 Mission U500 — Анализатор мочи
**Принцип**: рефлектометрия тест-полосок. До 500 тестов/час.
**Расходник**: **1 тест-полоска = 1 тест** (все 10-11 параметров). Упаковка 100 шт.
Жидких реагентов **НЕТ**. Дискретный подсчет полосок.
**Пробирка**: контейнер для мочи + пробирка коническая.

### 2.8 Vision Pro (YHLO) — Анализатор СОЭ
**Реагенты**: **НЕТ**. Работает без реагентов, напрямую из первичных ЭДТА-пробирок.
Результат за 20 минут. Корреляция с Вестергреном. Минимум 1.5 мл крови.
"Левый" прогон СОЭ не тратит реагенты — только ресурс прибора.

### 2.9 Wondfo OCG-102 — Оптический коагулометр (POCT)
**Принцип**: POCT, одноразовые картриджи. **Тесты**: PT-INR, APTT, FIB, ACT, TT.
**Расходник**: **1 картридж = 1 тест = 1 параметр**. Объем пробы 20 мкл цельной крови.
Картриджи дорогие (POCT-формат).

### 2.10 ORTHO Workstation — Иммуногематология
**Принцип**: колоночная агглютинация (CAT) на ID-MTS гель-картах.
**Тесты**: группа крови ABO, Rh(D), скрининг антител, совместимость.
**Расходник**: **1 гель-карта = 6 микроколонок** (покрывает 1 полный тест forward+reverse).
Карты дискретные, стоимость ~500-2000 тенге/карта.
**Пробирка**: ЭДТА (сиреневая).

### 2.11 Edan i15 — Анализатор газов крови (POCT)
**Параметры**: pH, pCO₂, pO₂, Na⁺, K⁺, Cl⁻, Ca²⁺, Glucose, Lactate, Hematocrit.
**Расходник**: **1 одноразовый картридж = 1 тест** (все параметры). Reagent Pack на ~100-200 тестов.
Газовый баллон **не нужен** — встроенные тонометры.
**Проба**: ~100 мкл капиллярной/артериальной крови.

### 2.12 Fluorecare — POCT иммунофлуоресцентный анализатор
**Принцип**: лазерная флуоресценция на одноразовых тест-картриджах.
**Тесты**: CRP, PCT, cTnI, NT-proBNP, D-Dimer, HbA1c, TSH и др.
**Расходник**: **1 картридж = 1 тест на 1 параметр**. Готовый к использованию.
Проба 10-75 мкл, результат за 3-15 мин.

### 2.13 iFlash 1800 (YHLO) — ИХЛ анализатор
**Принцип**: прямая хемилюминесценция. До 180 тестов/час. 112+ параметров.

| Расходник | Расход на тест |
|-----------|---------------|
| Позиция в тест-кассете | 1 (кассета = 50 тестов) |
| Substrate | ~50 мкл |
| Wash Buffer | ~1.5 мл |
| Reaction Cup (кювета) | 1 шт |
| Sample Tip (наконечник) | 1 шт |

Кассеты с **чипом** — анализатор точно знает оставшиеся тесты. Можно сверять с ЛИС.
**Пробирка**: сыворотка (красная/желтая крышка).

### 2.14 Ditreex RO-400 + Фильтр RO F4-4-10Gm
**НЕ анализатор** — система водоподготовки (обратный осмос).
Расходники: фильтры предварительной очистки (3-6 мес), мембрана (1-2 года), угольный фильтр (6-12 мес).

## 4. ПАРСИНГ ЛОГОВ — СПЕЦИФИКАЦИЯ

### 4.1 Applogs — паттерны
**Легитимный тест**: `SAMPLE! {barcode}` → `Sample request count: 1` → JSON с OrderResearchID
**"Левый" прогон**: `SAMPLE! {barcode}` → `THERE IS NO SR FOR Barcode {barcode}` → `samplerequests==0`
**Данные в JSON**: WorkPlaceID, AnalyzerId, ServiceID, ParameterID, ResultText, Test Mode

### 4.2 errors.xml — парсинг
Каждый `<Sample Code="X">` — прогон, не записанный в ЛИС. Сопоставить Code с ЛИС: если нет → "левый".
Содержит: Test Mode (CBC+DIFF/CBC), Blood Mode (W/P), Take Mode (O), все параметры.

### 4.3 Категоризация "левых"
| Категория | Критерий | Учет |
|-----------|---------|------|
| LEGITIMATE | Есть в ЛИС | Поток 1 |
| PROBABLE_RERUN | Нет в ЛИС, но ±30 мин есть легит. тест, похожие результаты | Сервисный |
| SUSPICIOUS | Нет в ЛИС, нет близкого легит. теста, реальные значения крови | "Левый" |
| WASH/TEST | Результаты ≈ 0 (WBC=0.01, RBC=0.00, HGB=0) | Сервисный |
| QC | IsOrderProductionControl: true | Сервисный |

**Пример из ваших логов**: Barcode=1 (WBC=0.01, RBC=0.00) → WASH/TEST. Barcode=2 → WASH/TEST.

## 5. СЕРВИСНЫЕ РАСХОДЫ (теоретический расчет)

### BC-5000 ежедневно (рабочий день):
Startup+Background+Shutdown+QC(3)+~5 standby = **~286 мл Dil + 6 мл DIFF + 1.2 мл LH**
За месяц (22 дня): ~6,300 мл Dil + 132 мл DIFF + 26 мл LH

### BS-240/430/600M ежедневно:
QC 2 уровня × ~15 тестов + промывка + prime ≈ **~7 мл суммарно + 10-20 мл wash**

### C3100 ежедневно:
QC 2 уровня на PT/APTT/FIB + промывка ≈ **~2-3 мл каждого реагента**
**+ потери**: растворенные лиофилизаты живут 4-8 часов → неиспользованные остатки = потеря

### iFlash 1800 ежедневно:
QC 2 уровня × 5-10 параметров = **10-20 кассетных позиций + кюветы + наконечники**

## 7. ПРОВЕРКА ЧЕСТНОСТИ ЛОГОВ

- Лаборант **отключит сетевой кабель** → лог не запишется, но gaps во времени видны
- **Рекомендация**: раз в месяц USB-выгрузка total count анализатора → сравнить с (Поток1 + Поток2 + сервисные). Если total count > суммы → были прогоны без логирования
- ЛИС-драйвер логирует ВСЕ полученные HL7-сообщения — лаборант не может отключить логирование

## 8. ИТОГ: ЧТО МОЖНО СЧИТАТЬ БЕЗ ЗАГЛЯДЫВАНИЯ В АНАЛИЗАТОР

| Задача | Источник | Точность |
|--------|---------|----------|
| Расход на легитимные тесты | ЛИС × нормы из вкладышей | ±5% |
| Сервисный расход | Теоретический расчет | ±15-20% |
| Кол-во "левых" прогонов | Applogs + errors.xml | Высокая (для интегрированных) |
| Стоимость "левых" | "Левые" × нормы × цена | ±10% |
| Расхождение | Теоретический остаток vs USB/склад | ±5% |
| Отключение от ЛИС | USB total count vs сумма потоков | Высокая при ежемесячной выгрузке |

**Всё считается программно**, аналитику показываются готовые цифры и расхождения.
